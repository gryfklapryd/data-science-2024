## 🎯 Tujuan Belajar Data Science

Melalui mata kuliah ini, saya memiliki beberapa target capaian pembelajaran, baik secara teoretis maupun praktis:

1. **Memahami Fundamental Data Science:**
   Menguasai siklus hidup data mulai dari pengumpulan data (*data gathering*), pembersihan data (*data cleaning/preprocessing*), hingga analisis data eksploratif (EDA).

2. **Menguasai Model Machine Learning:**
   Mempelajari algoritma dasar *Supervised* dan *Unsupervised Learning* (seperti Regresi Linier, Klasifikasi, dan *Clustering*) untuk membuat prediksi dan mengambil keputusan berbasis data.

3. **Integrasi Data Science dengan Web Application (End-to-End):**
   Memanfaatkan latar belakang saya di *full-stack development* untuk tidak hanya melatih model, tetapi juga men-deploy model *Machine Learning* tersebut (misalnya merancang API berbasis Python yang dapat dikonsumsi oleh *backend* Laravel dan divisualisasikan menggunakan antarmuka React/Tailwind).

4. **Visualisasi Data yang Interaktif:**
   Mampu menyajikan temuan data dan analitik secara visual agar mudah dipahami oleh *stakeholder* atau pengguna akhir.

5. **Penerapan pada Studi Kasus Nyata:**
   Menerapkan teknik pemrosesan data untuk memecahkan masalah sistem dunia nyata, seperti manajemen data infrastruktur atau dasbor analitik publik.
